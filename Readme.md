#32-bit PhantomJS Puppet Module
This puppet module setup PhantomJS on a 32-bit Ubuntu box

##Requirements
 None at all :-)

##Installation
Copy the module to the Puppet module folder

##Usage
Use `include phantomjs` in your manifest file, to run the installation
